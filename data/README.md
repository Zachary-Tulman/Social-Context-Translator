The social situation data used for the RAG in this project is stored in multiple .txt files sorted by social context.

The data format is as follows:

```
SITUATION:

RESPONSE:

---

SITUATION:

RESPONSE:

---
...
```

`SITUATION` explains a social situation where a person did not fully understand the social context surrounding the situation.
`RESPONSE` is an appropriate, neutral explanation of the situation and the social context behind everything that occurred. The response takes care to be gentle yet objective.
